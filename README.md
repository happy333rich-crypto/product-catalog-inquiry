# 產品電子型錄＋詢價清單

第一版為純前端網站，使用原生 HTML、CSS、JavaScript，讀取 `products.json` 顯示18項公開商品資料。

## 已完成

- 顯示18項商品，沒有圖片時顯示「商品圖片待補」
- 搜尋商品名稱、貨號、條碼
- 依品牌與商品分類篩選
- 加入詢價清單
- 增減箱數、手動輸入數量、移除商品、清空全部
- 使用 `localStorage` 暫存詢價清單與填寫中的聯絡資料
- 驗證店家名稱、聯絡人、聯絡電話必填欄位
- 產生繁體中文詢價文字
- 一鍵複製詢價內容，方便貼到 LINE
- 手機、iPad、電腦響應式版面
- 不顯示價格、進價、底價、毛利或內部備註

## 檔案

- `index.html`：網站頁面結構
- `style.css`：版面、色彩與手機響應式樣式
- `script.js`：商品載入、搜尋與篩選
- `cart.js`：詢價清單、數量調整與暫存
- `inquiry.js`：顧客資料、詢價文字與複製功能
- `products.json`：第一批18項公開型錄商品資料
- `PROJECT_BRIEF.md`：第一版需求與驗收項目

## 預覽方式

網站會用 `fetch()` 讀取 `products.json`，因此不要直接雙擊 `index.html` 開啟。請使用其中一種方式：

1. 發布為靜態網站後開啟網站網址。
2. 電腦端在專案資料夾啟動本機伺服器，例如：

```bash
python -m http.server 8000
```

再開啟 `http://localhost:8000`。

## 資料安全

目前聯絡資料只存在使用者自己的瀏覽器 `localStorage`，不會上傳到伺服器。公開商品資料不包含進價、毛利、底價與內部促銷條件。
